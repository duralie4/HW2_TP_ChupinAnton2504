# HW2_TP_ChupinAnton2504

Домашнее задание 2 по Технологиям программирования, БИ, 2026.
Модель рецептов и списка покупок: классы `Ingredient`, `Recipe`,
`DietaryRecipe`, `ShoppingList` и тесты к ним.

## Классы

- **`Ingredient`** — ингредиент рецепта: название, количество, единица
  измерения. Количество должно быть положительным. Два ингредиента равны,
  если совпадают название и единица измерения.

- **`Recipe`** — рецепт блюда: название и список ингредиентов. Можно
  добавлять ингредиенты (дубликаты объединяются по названию и единице),
  масштабировать количества на коэффициент и узнавать число ингредиентов.

- **`DietaryRecipe`** — рецепт с диетической категорией (например, веган
  или без глютена), наследуется от `Recipe`. При масштабировании категория
  сохраняется, а в выводе добавляется префиксом: `[веган] Пицца`.

- **`ShoppingList`** — список покупок из одного или нескольких рецептов.
  Принимает рецепт с числом порций, удаляет ингредиенты по названию рецепта,
  суммирует одинаковые ингредиенты из разных рецептов. Списки складываются
  через `+`.
